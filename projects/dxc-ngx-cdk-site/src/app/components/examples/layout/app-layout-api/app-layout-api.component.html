This component has no props. It has children and it follows the design guidelines available .

Everything in this component will be displayed as a header, at the top of the screen. If you want to show a , as a shortcut, you can directly use it without DxcApplicationLayoutHeader. This is optional and if it is not specified, the DxcHeader will be shown by default.

Everything in this component will be displayed as content inside a sidenav at the left. This is optional and if not specified, the sidenav will not be shown. Check for more information regarding its props. This also applies to the themming.

This component has to be a child of DxcApplicationLayoutMain. Everything in this component will be displayed as a footer, at the bottom of the screen. If you want to show a , as a shortcut, you can directly use it without DxcApplicationLayoutFooter. This is optional and if it is not specified, the DxcFooter will be shown by default.
